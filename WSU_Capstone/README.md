These are the data analysis codes I created for my senior design project while at Washingtion State University.

For our project, we used a sonar sensor to scan large pools. However, the software that came with the sensor did not have the best clairity. So, we decided to use python to plot our data. It also could only produce a 2D scan. This could splices the different scans together to create a full 3D scan.

One of the problems we had was that the sonar only outputted hexadecimal files. So, we first had to convert them to csv files, then plot them.
